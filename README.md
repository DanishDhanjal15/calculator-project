# calculator-project


A sleek, modern calculator built with vanilla HTML, CSS, and JavaScript. Features a clean interface with smooth animations and responsive design.

## Features

- *Basic Operations*: Addition, subtraction, multiplication, and division
- *Advanced Functions*: 
  - Clear display (C)
  - Backspace (⌫) 
  - Sign toggle (+/-)
  - Decimal point support
- *Modern UI*: Dark theme with gradient backgrounds and smooth hover effects
- *Responsive Design*: Works on desktop and mobile devices
- *Error Handling*: Graceful error display for invalid calculations

## How to run

### Quick Start
- Open h.html in any modern browser
- No setup or installation required

### Local Development Server (Recommended)
For better development experience, use a local server:

*Python 3:*
bash
python3 -m http.server

Then visit http://localhost:8000/h.html

*Node.js (if you have it):*
bash
npx serve .

Then visit the provided URL

## Usage

1. *Basic Calculations*: Click number buttons and operators to build expressions
2. *Clear*: Press 'C' to reset the calculator
3. *Backspace*: Press '⌫' to remove the last character
4. *Sign Change*: Press '+/-' to toggle between positive and negative numbers
5. *Calculate*: Press '=' to see the result

## Technical Details

- *Pure JavaScript*: No frameworks or dependencies
- *CSS Grid*: Responsive button layout
- *Modern CSS*: Flexbox, gradients, and smooth transitions
- *Error Handling*: Try-catch blocks for invalid expressions
- *Accessibility*: Keyboard-friendly with proper focus states
